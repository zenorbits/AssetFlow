# AssetFlow

AssetFlow is a web application for managing organizational assets — track assets, view analytics, and keep a full audit trail, all behind secure authentication.

## Features

- **Asset Management** — Add, update, categorize, and track assets throughout their lifecycle
- **Analytics Dashboard** — Visualize asset usage, distribution, and trends at a glance
- **Audit Logs** — Full history of who changed what and when, for accountability and compliance
- **Authentication** — Secure sign-in/sign-up flow to protect access to asset data
- **Smooth UI Animations** — Polished, animated interactions powered by Framer Motion
